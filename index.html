Git Init:

  • First initialize a Git Repository

Git Add:

  • Add a file in git Repository Use "Git Add Filename" command.

  • Use a dot (.) instead of a filename to add all the files in the folder into git Repository

Git Commit:

  • To commit a change in Git use command "Git Commit -m"
  • Here type any message in double quotes to show information about your commit

Git Push:

  • To push changes onto the remote repository use "git push" command

Git Status:

  • Git status tells all about the status of the repository

this change is made online

Editing Online:

  • You can edit and commit changes in a file online

Git Pull:

  • To save the changes that are made online into our local repository we use Git Pull

  • To create a branch we use the following commands

  • To create new branches and to switch between branches use the checkout command
