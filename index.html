Git Init:

  • First initialize a Git Repository

Git Add:

  • Add a file in git Repository Use "Git Add Filename" command.

  • Use a dot (.) instead of a filename to add all the files in the folder into git Repository

Git Commit:

  • To commit a change in Git use command "Git Commit -m"
  • Here type any message in double quotes to show information about your commit

Git Push:

  • To push changes onto the remote repository use "git push" command
